Enhanced Syntax Highlighting Features

  • GraphQL Keywords: Red highlighting for query, mutation, subscription, fragment
  • GraphQL Types: Purple highlighting for type names (Missionary, ID, String, etc.)
  • GraphQL Fields: Blue highlighting for field names and arguments
  • GraphQL Variables: Orange highlighting for variables like $missionaryNumber
  • GraphQL Strings: Dark blue highlighting for string literals
  • JSON Keys: Blue highlighting for object property names
  • JSON String Values: Dark blue highlighting for string values
  • JSON Numbers: Blue highlighting for numeric values
  • JSON Booleans/Null: Appropriate color coding for special values
  • Smart Indentation: Proper formatting and indentation for readability
  • Copy to Clipboard: One-click copying of code or responses
  • Fullscreen Mode: Expand editors for better visibility
  • Dark/Light Theme: Automatic theme switching based on system preferences

How it Works

The enhanced syntax highlighting uses CodeMirror 6 with custom highlight styles that:

  • Parse GraphQL syntax to identify keywords, types, fields, and variables
  • Parse JSON to distinguish between keys, values, and data types
  • Apply consistent color coding that matches popular code editors
  • Support both light and dark themes automatically
  • Provide interactive features like line numbers, bracket matching, and selection highlights
